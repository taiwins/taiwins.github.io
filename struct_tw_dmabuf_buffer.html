Detailed Documentation ¶

a dma param/buffer object

param resource is created at dmabuf.create_params. Mutiple planes can be added to this paramter before dmabuf_param.create or dmabuf_param.create_immed is called. After that param_resource is gone and this tw_dmabuf_buffer represents a wl_buffer.

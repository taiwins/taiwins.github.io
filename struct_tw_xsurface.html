Detailed Documentation¶

tw_xsurface implements a tw_desktop_surface for xcb_window

Xwayland creates new wl_surface for every xcb window by sending a wl_surface id. Here the job is mapping the xcb window logic to the tw_desktop_surface API.

xcb window would send a MapRequest event for mapping the surface on the window manager, this usually happens before we actually receive the wl_surface id.
